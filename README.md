# Divine Mandate Bible Institute (DIMABIN) - Responsive Web Portal

A premium, highly responsive, mobile-first academic website and administrative portal tailored for the **Divine Mandate Bible Institute (DIMABIN)**, an interdenominational citadel of learning.

This repository is optimized for instant deployment across major hosting platforms with direct live Firestore database integrations and comprehensive student/lecturer/admin portal configurations.

---

## 🚀 Key Features

- **100% Mobile First & Responsive Design**: Styled utilizing fluid grids, container queries, and Material 3 design tokens. Looks beautiful on mobile, foldables, tablets, and wide desktop views.
- **Academic Programs & Foundations**:
  - **Home**: Introducing DIMABIN courses, Rector's desk message, core Christian principles, and student portals.
  - **About Us**: Detailed core mandate, historical foundations under Sorinola J.O., and operational objectives.
  - **Admissions**: Live theological program catalog, requirements, fees, and enrollment workflow.
  - **Student Portal**: Access modules, grading matrices, continuous assessment, and exams.
  - **Lecturer Portal**: Manage grades, classes, and spiritual facilitation.
  - **Admin Portal**: Fully integrated admissions processing engine, student registries, and system activity logs.
- **Dynamic Live DB Interactivity (`firebase-core.js`)**: Robust integration with Firebase Auth and Firestore for live database operations, activity logging, and portal login.
- **Polished Visual Assets**: Standard vector academic emblems (the official DIMABIN crest), high-resolution campus assets.
